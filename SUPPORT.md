# Supporto

## Trovare aiuto

- **Errori nei contenuti** → apri una issue nella repository specifica della materia (es. `2526_programmazione`).
- **Problemi con il sito** → apri una issue in [`stangauni.github.io`](https://github.com/StangaUni/stangauni.github.io).
- **Domande generali** → usa le Discussions della repository più pertinente.

## Cosa non è supportato

Questo progetto fornisce appunti personali a scopo didattico. Non è un servizio di tutoraggio: non garantisco risposte a domande di studio o chiarimenti sugli argomenti trattati.

Per supporto accademico, rivolgiti ai docenti, ai tutor ufficiali del corso, o ai canali di studio del tuo corso di laurea.
