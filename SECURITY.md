# Security Policy

Questo progetto contiene esclusivamente documentazione (file Markdown, MDX, PDF) e codice front-end statico. Non gestisce dati sensibili, autenticazioni, o backend.

## Segnalare un problema

Se identifichi un problema di sicurezza nel **sito** [stangauni.github.io](https://stangauni.github.io) (es. dependency con vulnerabilità note, script injection nel rendering MDX), apri una issue nella repository [stangauni.github.io](https://github.com/StangaUni/stangauni.github.io) descrivendo il problema.

Per problemi di natura particolarmente sensibile, contatta il maintainer direttamente tramite i contatti indicati nel profilo GitHub.
