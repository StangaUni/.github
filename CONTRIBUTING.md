# Partecipazione alla community

Queste repository contengono appunti personali redatti come strumento di studio. Non è prevista alcuna forma di contribuzione diretta ai contenuti: le **pull request sono disabilitate**.

## Come interagire

### Hai trovato un errore?

Apri una issue usando il template **Errore nei contenuti**. L'errore verrà valutato e, se confermato, corretto dal maintainer.

### Manca un argomento?

Apri una issue usando il template **Richiesta argomento**. Tieni presente che gli appunti seguono il programma del corso e le aggiunte dipendono esclusivamente dalla disponibilità del maintainer.

### Hai una domanda o vuoi discutere?

Usa la sezione **Discussions** della repository pertinente o dell'organizzazione. È il posto giusto per confrontarsi sugli argomenti trattati, chiedere chiarimenti o condividere risorse aggiuntive.

### Problema con il sito?

Apri una issue usando il template **Problema con il sito** nella repository [`stangauni.github.io`](https://github.com/StangaUni/stangauni.github.io).
